# Teste prático CoreBiz

## Ferramentas utilizadas
    - HTML
    - CSS (Stylus)
    - JC (React)

## Requisitos:
    - Node >= 14.0.0 
    - npm >= 5.6 

## Scripts para rodar o projeto:

    Instalar dependências do projeto:
    $ npm i

    Rodar projeto:
    $ npm start

    Monitorar e processar Stylus:
    $ npm run gulp
